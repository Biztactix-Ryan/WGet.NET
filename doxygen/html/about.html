Docs

The documentation was generated with the help of Doxygen.

The base style, dark/light mode toggling and other UI improvements are done with doxygen-awesome-css.
There are additional custom styles and scripts, and the doxygen-awesome-darkmode-toggle.js script was modified for this project.

Custom html pages, like this "About" page, are done with the doxygen-custom-page-injector, which was initially created for this project.
